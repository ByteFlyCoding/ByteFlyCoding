1. 两数之和

  • 法一：

    func twoSum(nums []int, target int) []int {
	var tmpMap = map[int]int{}
	for i, v := range nums {
      tmpMap[v] = i
	}
  
	for i, v := range nums {
      if pos, ok := tmpMap[target - v]; ok && pos != i{
          nums[0] = i
          nums[1] = pos
          break;
      }
	}
	return nums[0:2]
}

    复杂度分析：

    1. 时间复杂度：O(n)

    2. 空间复杂度：O(n)

  • 法二：

    func twoSum(nums []int, target int) []int {
	tmpMap := map[int]int{}

      if pos, ok := tmpMap[target-v]; ok {
          nums[0] = pos
          nums[1] = i
          break
      }
      tmpMap[v] = i
	}
	return nums[0:2]
}

    复杂度分析：

    1. 时间复杂度：O(n)

    2. 空间复杂度：O(n)

26. 删除有序数组中的重复项

  • 法一：

    func removeDuplicates(nums []int) int {
	for i := len(nums) - 1; i > 0; i-- {
      if nums[i] == nums[i-1] {
          nums = append(nums[:i-1], nums[i:]...)
      }
	}
	return len(nums)
}

    复杂度分析：

    1. 时间复杂度：O(n)

    2. 空间复杂度：O(1)

  • 法二：

    func removeDuplicates(nums []int) int {
	res := 1
	for i := 1; i < len(nums); i++ {
      if nums[i] != nums[i-1] {
          nums[res] = nums[i]
          res++
      }
	}
	return res
}

复杂度分析：

  1. 时间复杂度：O(n)

  2. 空间复杂度：O(1)

136. 只出现一次的数字

  • 法一：

    func singleNumber(nums []int) int {
	tmpMap := make(map[int]int)
	for _, value := range nums {
      tmpMap[value] += 1
	}
	for i, _ := range tmpMap {
      if tmpMap[i] == 1 {
          return i
      }
	}
	return 0
}

复杂度分析：

  1. 时间复杂度：O(n)

  2. 空间复杂度：O(n)

  • 法二：

    用位运算 利用异或运算性质：

    1. 任何数和 0 做异或运算，结果仍然是原来的数
    2. 任何数和其自身做异或运算，结果是 0
    3. 异或运算满足交换律和结合律
    func singleNumber(nums []int) int {
    single := 0
    for _, num := range nums {
        single ^= num
    }
    return single
}

复杂度分析：

  1. 时间复杂度：O(n)

  2. 空间复杂度：O(1)
